# Ex-5e — EMI Calculator

Simple client-side EMI calculator.

How to use

1. Open `Ex-5e/index.html` in a browser.
2. Enter Loan Amount (P), Annual Interest Rate (%) and Tenure (years). The results update as you type (onkeyup).

Notes
- EMI formula used: EMI = [P * R * (1+R)^N] / [(1+R)^N - 1], where R = annual/12/100 and N = years*12.
- Values are shown in Indian Rupees (₹).
