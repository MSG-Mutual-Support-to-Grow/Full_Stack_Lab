# Ex-7b — Budget Calculator (jQuery)

Simple client-side budget calculator using jQuery.

How to use
- Open `Ex-7a/index.html` in a browser.
- Enter the total amount and percentage values for Room Rent, Accessories, Emergency and Saving.
- Click "Evaluate My Money". Percentages must add up to 100%, otherwise an alert will appear.

Files
- `index.html` — UI and jQuery include.
- `styles.css` — minimal styling.
- `script.js` — evaluation logic using jQuery.

Notes
- This is a simple front-end demo. No server or build step required.
